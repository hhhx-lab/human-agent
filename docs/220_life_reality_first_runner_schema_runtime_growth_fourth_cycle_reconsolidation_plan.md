## 本层定位

`220` 是第四轮 post validation seed 后的 reconsolidation 层。`219` 已经把 validation findings 压成 seed，本层把 seed 放入再巩固：

```text
post validation seed bundle -> seed intake -> fast/slow memory partition -> lability window -> online replay/offline dream consolidation/targeted reactivation/interleaved replay -> language relationship/dream-pain-regret/self-growth/readiness/life-target reconsolidation -> arbitration -> receipts/digest/dashboard/stage -> post reconsolidation observation handoff
```

reconsolidation 不是单纯保存。被唤起的记忆、关系、承诺、痛苦、后悔、自我结构和技能路线会短暂进入可塑窗口；如果没有 CoreContinuityGuard、DreamFactGate、anti-forgetting replay、relationship timeline 和 rollback/quarantine gate，这个窗口会破坏旧自我。`220` 的核心任务是让新 seed 被吸收，同时让旧自我仍然能被回忆、被表达、被修复、被继续成长。
