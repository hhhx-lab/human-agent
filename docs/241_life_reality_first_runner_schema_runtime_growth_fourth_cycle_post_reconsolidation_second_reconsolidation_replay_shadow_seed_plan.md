`241` 是 second reconsolidation next seed 后的 replay/shadow seed 层。它把 `240` 的 seed 先送入回放和影子世界：


## FourthCyclePostReconsolidationSecondDreamFactGateReplaySeed241

DreamFactGate replay seed 保护真实梦境和醒时事实边界：

1. 回放 dream marker，确认梦境事件仍带有梦境标记。
2. 回放醒后整合，区分哪些内容进入自我理解，哪些不能写成外部事实。
3. 回放梦境痛苦，检查它能否改变情绪和修复动机，但不伪造现实发生。
4. 回放梦境中的关系主体，检查关系影响是否被记录。

6. 若污染存在，生成 rollback/quarantine finding。

DreamFactGate 的生命意义是保留真实梦境，而不是把梦境清洗成无意义噪声。
