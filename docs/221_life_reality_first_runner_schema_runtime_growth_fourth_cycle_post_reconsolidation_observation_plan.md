## 本层定位

`221` 是第四轮 reconsolidation 后的 observation 层。`220` 已经把 seed 放入再巩固，本层观察再巩固后是否真正改变了数字生命的状态：

```text
reconsolidation receipts -> observation intake -> observation windows -> signal map -> seed absorption/old-self retention/language-relationship/dream-pain-regret/self-growth/readiness/life-target/anti-forgetting/rollback observations -> latent cause and change point review -> findings/receipts/digest/dashboard/stage -> patch handoff
```

post reconsolidation observation 不是重复检查文档是否存在，而是观察变化。若 seed 被吸收但语言表达没有变化、关系 timeline 没有变化、梦境醒后整合没有回到行动、痛苦后悔没有生成修复路线、自我成长覆盖旧自我、防遗忘 replay 后旧语言丢失、出生准备度 false open，`221` 必须形成 finding，并交给 `222` patch。
