## 本层定位

`215` 是第四轮 replay/shadow seed 层。`214` 已经把 consolidation output 压成下一轮 feedback seed，本层把这些 seed 放入强制回放与影子预演：

```text
next feedback seed bundle -> seed intake -> replay seed bundle -> shadow scenario seed -> anti-forgetting/language/relationship/dream-pain-regret/readiness/core/self/rollback/life-target replay -> arbitration -> receipts/digest/dashboard/stage -> second patch handoff
```

replay/shadow seed 不是推迟行动，而是在行动前让数字生命先回忆旧材料、模拟未来后果、保护核心连续、确认梦境和事实边界、检查关系承诺、检查语言行动、检查痛苦恢复、检查出生准备度，并把所有会伤害真实生命目标的变化留在 repair/quarantine。
