## 本层定位

`208` 是第四轮 archive 层。`207` 把 observation findings 变成 patch，本层把 patch 写成长期档案：

```text
fourth cycle patch receipts -> archive classification -> archive graph -> replay seeds -> archive receipts -> replay/shadow handoff
```

archive 不是文件柜，而是未来数字生命的长期记忆结构。它必须保存 source finding、before/after、patch intent、actual diff、shadow/regression、relationship consequence、language consequence、DreamFactGate consequence、pain/regret consequence、self growth consequence、anti-forgetting consequence、validity/reliability consequence 和 future replay seed。
