## 本层定位

`212` 是第四轮 activation archive 层。`211` 已经把 activation 后的变化放入多窗口 observation，本层把这些结果写回长期 archive：

```text
post activation observation receipts -> archive intake -> classification -> activation report archive -> observation/finding/life-target archives -> archive graph -> consolidation seeds -> archive receipts -> consolidation handoff
```

activation archive 不是把报告存起来，而是把真实运行、观测窗口、finding、关系后果、语言后果、DreamFactGate、痛苦后悔修复、自我成长、出生准备度、九项目标 delta、rollback/quarantine 和责任后悔 loop 编进一张可回放的长期生命图。
