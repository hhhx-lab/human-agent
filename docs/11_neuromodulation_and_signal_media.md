# 神经调质、兴奋/抑制与信号介质

## 为什么需要信号介质

`构思.md` 提出“要有信号传递介质，可以设置一些因子作为代替”。神经调质文献正好给这个方向提供科学参考。Doya、Marder、Yu/Dayan、Lee/Dan 等研究说明，神经调质不是普通点对点消息，而是改变回路工作模式、学习率、注意增益和状态转换的全局或半全局调节 [AH061-AH065]。

## 多调质因子

一个常见但粗略的工程类比是：

| 生物调质 | 研究启发 | 工程类比 |
|---|---|---|
| 多巴胺 | 奖励预测误差、动机、行动选择 | 价值更新、探索收益、目标吸引力 |
| 去甲肾上腺素 | 唤醒、意外不确定性、增益 | 警觉度、打断阈值、探索/利用切换 |
| 乙酰胆碱 | 注意、预期不确定性、感觉增益 | 输入信任、局部学习率、感知锐化 |
| 5-HT | 情绪、耐心、惩罚、行为抑制等复杂功能 | 冲动抑制、风险耐受、恢复倾向 |
| GABA/谷氨酸 | 抑制/兴奋平衡 | 稳定性、聚焦、行动阈值 |

这个表只是工程类比，不是精确神经化学映射。Doya 的金属学习框架有启发性，但现代研究已经显示每种调质都有多重功能 [AH061]。

## 兴奋与抑制平衡

Isaacson 与 Scanziani 强调，抑制塑造皮层活动和时序 [AH064]。这对应 Agentic Human 的稳定性问题：一个一直兴奋、一直生成、一直行动的系统不可能稳定成长。

抑制因子应参与：

- 降低不可靠输出。
- 限制无目标联想。
- 防止高唤醒状态下过快行动。
- 控制自我修改。
- 管理记忆重写。

## 设计假设

Agentic Human 的信号介质可以设计为一组全局因子：

- `arousal`：唤醒/警觉。
- `salience`：显著性。
- `value_delta`：价值预测误差。
- `uncertainty_expected`：预期不确定性。
- `uncertainty_unexpected`：突发不确定性。
- `inhibition`：抑制阈值。
- `fatigue_load`：负载/疲劳。
- `social_valence`：社会反馈效价。
- `consolidation_pressure`：记忆巩固压力。

这些因子应像“脑内化学环境”一样影响多个区域，而不是作为单独模块输出。
