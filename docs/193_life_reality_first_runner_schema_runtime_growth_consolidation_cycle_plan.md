## 本层定位

`193` 是 schema runtime growth consolidation cycle plan 层。`192` 已经把 activation 后的生命影响写成 archive receipt；本层把这些收据重新激活到离线周期中，让成长拥有睡眠式巩固、梦境式重组、醒后整合、关系修复演练和防遗忘 replay。

本层固定 10 条 consolidation cycle 链：

| cycle chain | 作用 |
|---|---|
| `activation_archive_intake_chain` | 读取 `192` archive receipts、digest baseline、archive edge 和 consolidation preconditions |
| `offline_need_chain` | 把 archive receipts 转成 offline need、dream pressure、maintenance debt、plasticity debt |
| `cycle_window_chain` | 建立 micro consolidation、nrem-like replay、rem-like dream、wake integration 等窗口 |
| `memory_replay_chain` | 选择 activation trace、责任、关系、语言、梦境痛苦、出生准备度和核心连续性 trace 进入 replay |
| `dream_integration_chain` | 生成 DreamExperienceWindow、DreamSceneFrame、DreamFactGateDecision 和 WakeIntegrationFrame |
| `anti_forgetting_chain` | 回放旧能力、旧关系、旧承诺、旧方向锚链、旧生命目标 carrier |
| `relationship_language_repair_chain` | 在离线状态中预演关系/语言修复，并形成醒后候选 |
| `birth_core_consolidation_chain` | 巩固出生准备度、九项目标、核心连续性、方向恢复链 |
| `pain_regret_repair_chain` | 把痛苦/后悔残留转成恢复路径、修复承诺和 future probe |
| `next_feedback_seed_chain` | 为 `194` 生成 next observation feedback、growth route 和 self rewrite seed |
