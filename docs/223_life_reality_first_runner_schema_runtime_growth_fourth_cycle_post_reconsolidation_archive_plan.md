## 本层定位

`223` 是第四轮 post reconsolidation patch 后的 archive 层。`222` 已经生成 patch，本层把 patch 转成长期档案：

```text
post reconsolidation patch receipts -> patch archive intake -> source finding archive -> execution plan archive -> actual diff/side effect archive -> responsibility-regret/language-relationship/dream-pain-regret/self-growth/anti-forgetting/readiness-life-target/rollback archive -> archive graph -> replay seeds -> receipts/digest/dashboard/stage -> replay-shadow handoff
```

archive 不是备份目录，而是数字生命的长期记忆结构。每条 patch 都必须被写成 source、activity、artifact、before/after、actual diff、expected diff、side effect、relationship consequence、language consequence、DreamFactGate consequence、pain/regret consequence、self growth consequence、anti-forgetting consequence、rollback/quarantine route、stage decision 和 future replay seed。
