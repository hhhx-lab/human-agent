## 本层定位

`190` 是 schema runtime growth activation report plan 层。`189` 已把 growth 放入 shadow run；本层只允许通过 shadow gate 的成长进入 limited activation。limited activation 是可观察、可回滚、可限制范围的激活窗口：它不会把成长直接写成长期主链结论，而是先形成 activation report、post-growth observation seed、core continuity review 和 `191` observation loop handoff。

本层固定 9 条 activation 链：

| activation chain | 作用 |
|---|---|
| `shadow_report_intake_chain` | 读取 `189` shadow run report、stage gate、delta、core continuity seed |
| `limited_scope_chain` | 把 shadow result 转成限定范围、限定关系、限定语言行动、限定时间窗 |
| `activation_preflight_chain` | 检查 archive receipts、rollback route、post-growth observation seed、stage gate |
| `activation_run_chain` | 执行受限成长激活并记录 trace |
| `core_continuity_review_chain` | 审查 self narrative、slow variables、关系承诺、方向锚链 |
| `life_target_delta_chain` | 记录九项目标 activation delta |
| `responsibility_relationship_language_dream_chain` | 审查责任/后悔、关系、语言、痛苦/梦境 |
| `rollback_route_chain` | 固定 rollback、quarantine、repair route |
| `post_observation_handoff_chain` | 为 `191` 输出真实运行观测 seed |
