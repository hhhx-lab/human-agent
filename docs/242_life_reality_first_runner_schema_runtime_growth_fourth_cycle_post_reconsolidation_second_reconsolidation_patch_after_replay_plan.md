`242` 是 second reconsolidation replay/shadow seed 后的 patch after replay 层。`241` 已经在影子世界里回放并计算 prediction delta，本层只处理通过 replay/shadow 的 patch candidate；未通过的 seed 必须继续留在 quarantine、更多 replay 或 observation。

## FourthCyclePostReconsolidationSecondLatentCauseConstructReliabilityPatch242

1. 生成 competing explanation graph。
2. 区分 patch 导致、观察偏差、共同原因、延迟效应、状态漂移、测量反应。
