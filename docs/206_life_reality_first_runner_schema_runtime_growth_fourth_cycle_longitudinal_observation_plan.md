## 本层定位

`206` 是第四轮长期 observation 层。`205` 把 `204` findings 压成第四轮 seed，本层把 seed 放回时间中观察：

```text
fourth cycle seeds -> observation windows -> signal map -> longitudinal probes -> findings/receipts -> fourth cycle patch handoff
```

本层必须区分四类变化：真实成长、短期状态、测量噪声和回退。只有能跨时间窗、跨关系场景、跨语言事件、跨梦境/醒后整合、跨痛苦修复和跨自我改写保持结构的变化，才能进入 `207` patch。
