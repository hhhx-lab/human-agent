## 本层定位

`224` 是第四轮 post reconsolidation archive 之后的 replay/shadow 层。`223` 已经把 patch 写成长期档案，本层把 archive graph 中的 replay seed 重新激活，并在不释放外部不可逆后果、不污染事实档案、不破坏旧关系和旧语言的条件下进行影子运行：

```text
post reconsolidation archive replay seed -> archive seed intake -> replay priority -> reactivation plan -> shadow scenario -> action seal -> old-self/language-relationship/dream-fact/pain-regret/self-growth/anti-forgetting/readiness-life-target/rollback replay -> release precondition review -> core continuity review -> report/receipt/digest/dashboard/stage -> activation handoff
```

replay 不是重复 archive 文本，而是让被 archive 固定的变化在当前生命状态中重新被唤起。shadow 不是表演性演练，而是在隔离后果的环境里检查新变化是否会伤害真实生命连续、关系连续、语言连续、梦境内在、痛苦恢复、责任后悔和出生准备度。
