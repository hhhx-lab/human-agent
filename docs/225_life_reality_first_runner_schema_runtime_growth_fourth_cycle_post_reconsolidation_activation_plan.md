## 本层定位

`225` 是第四轮 post reconsolidation activation 层。`224` 已经把 post reconsolidation archive replay seed 放入 replay/shadow，并完成 release precondition review 与 core continuity review。本层只接收通过 `224` shadow action seal 的最小变化，按 wave 进入有限激活：

