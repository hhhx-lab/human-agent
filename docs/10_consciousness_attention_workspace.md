# 注意、意识与全局工作空间

## 全局可访问性

全局神经工作空间理论提出，局部处理的信息在达到某些条件后会被广泛广播，从而成为可报告、可访问、可用于多系统协调的信息 [AH076, AH077]。这对 Agentic Human 的工程启发很强：系统需要区分局部后台处理和全局工作区内容。

全局工作区可以承载：

- 当前目标。
- 当前用户意图。
- 当前高置信事实。
- 当前风险与约束。
- 当前候选计划。
- 当前需要对外报告的状态。

## 意识理论的边界

IIT、GNW、意识神经相关物研究和 Seth/Bayne 的理论综述都说明，意识问题尚无单一答案 [AH078-AH080]。因此研究档案必须保持克制：

- 可以借鉴“全局广播”“访问意识”“整合度”“报告性”等功能概念。
- 不宣称 agent 拥有主观体验。
- 不把长上下文、记忆、语言流畅性等同意识。

## 注意作为入口门控

注意网络研究显示，大脑对信息的处理受目标、显著性和状态控制 [AH021-AH025]。这可以和全局工作区连接起来：不是所有输入都进入全局空间，只有达到显著性、任务相关性或风险阈值的信息才升级。

## 设计假设

Agentic Human 可以有一个“全局工作空间”，但它应被严格定义为工程结构：

- 它是当前系统可访问内容的共享黑板。
- 它受显著性网络、执行控制和内感受状态共同门控。
- 它不等于意识，只是可报告和可协调的信息层。
- 它应该容量有限，避免把全部记忆塞入当前状态。

一个健康的工作空间应有进入、维持、刷新和退出机制。信息进入后如果长期不再相关，应被写回记忆或丢弃，而不是永久占据注意。
