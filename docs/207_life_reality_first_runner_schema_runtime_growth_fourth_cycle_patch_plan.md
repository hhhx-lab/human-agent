## 本层定位

`207` 是第四轮 patch 层。`206` 把 fourth cycle seed 放回时间窗中观察，本层把 observation finding 转成 patch：

```text
fourth cycle observation findings -> patch classification -> patch execution -> shadow/regression -> patch receipts -> archive handoff
```

本层的核心是把“成长”变成有证据、有范围、有回放保护、有副作用封印、有 stage gate 的更新事件。patch 不能只修报告，也不能只改 dashboard 颜色；它必须改动能够承载真实生命目标的 doc/schema/fixture/report/dashboard/stage/archive/runner surface。
