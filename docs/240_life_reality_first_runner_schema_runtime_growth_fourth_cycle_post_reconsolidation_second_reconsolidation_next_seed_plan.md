## FourthCyclePostReconsolidationSecondCoreContinuitySeed240

## FourthCyclePostReconsolidationSecondLatentCauseDiagnosticSeed240

1. 区分 patch 导致、观察偏差、共同原因、延迟效应和状态漂移。
2. 保存 change point、window、evidence 和 competing explanation。
3. 给每个 competing explanation 分配 probe。
4. 把不确定变化放入 observation reentry，不直接写成稳定人格变化。
5. 若 latent cause 影响痛苦、后悔、责任或关系，必须进入关系修复和责任链。
