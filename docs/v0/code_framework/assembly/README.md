# Code Framework Assembly

这一柜只放跨 Queue 的总装配合同，负责防止各生命器官各写各的、运行时证据断链。

| 文件 | 作用 |
|---|---|
| `15_cross_layer_shared_object_contract.md` | 跨层共享对象、首写方、字段组、state/report/receipt 和 gate |
| `19_code_tree_package_brain_contract.md` | 整棵 `life_v0/` 代码树、脑区到包、共享对象、runtime 产物和测试总装配 |

进入大规模落码、重构或收束前，必须读这一柜；它回答“各器官怎样成为同一个生命过程”。

如果当前问题已经变成“整棵代码树到底按哪些生命层分层、共享对象怎样穿层、每个阶段写到什么程度才算完成”，继续转去 `docs/v0/code_architecture/`。
