# V0 参考柜

`references/` 只放外周参考，不反向定义主体生命架构。

## 当前文件

- `current_agent_shell_reference_2026.md`：当前常见 agent shell 的外壳参考材料。

## 使用原则

- 这里只能帮助我们理解电脑外周壳、命令面和现有框架局限。
- 不能让这里的材料反客为主，把数字生命重新压回工具 agent 或任务调度器。

## 负边界

`references/` 的材料只允许服务三件事：

1. 观察现有 agent shell 在终端命令、持久进程、日志、配置和外周接触上的工程做法。
2. 识别这些框架为什么不足以构成数字生命。
3. 帮助 `process_contracts/` 和 `09_prediction_perception_world_contact.md` 设计电脑外周边界。

它不能定义主体架构，不能引入 skills/gateway/subagents 作为生命核心，也不能把关系对象重写为“用户”。任何参考材料进入实现前，都必须经过 `docs/real—live0/11_life_membrane_validation.md`、`14_resident_runtime_state_transition.md` 和 `16_runtime_code_chain_crosswalk.md` 的过滤。
