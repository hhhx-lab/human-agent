## 本层定位

`222` 是第四轮 post reconsolidation observation 后的 patch 层。`221` 已经观察再巩固后的真实变化，本层把 observation findings 转成 patch：

```text
post reconsolidation observation findings -> finding intake -> patch classification -> priority/control arbitration -> seed absorption/old-self/language-relationship/dream-pain-regret/self-growth/readiness/life-target/anti-forgetting/rollback/latent-cause patches -> execution plan -> receipts/digest/dashboard/stage -> archive handoff
```

patch 不是任意改动，而是由 finding 驱动的生命修复。每个 patch unit 必须知道：来自哪条 observation finding、修复哪条真实生命目标、影响哪些旧自我材料、是否触发语言关系改变、是否触发梦境痛苦后悔修复、是否需要防遗忘回放、失败时如何 rollback/quarantine、完成后交给哪个 archive edge。
