## 本层定位

`218` 是第四轮 second validation 层。`217` 已经把 second patch 写回 archive，本层验证整个二次成长段是否闭合：

```text
second archive receipts -> archive intake -> validation graph -> observation/direction/readiness/life-target/language-relationship/dream-pain-regret/self-growth/anti-forgetting/rollback validation -> construct validity/reliability review -> findings/receipts/digest/dashboard/stage -> next seed handoff
```

validation 不是给 archive 盖章，而是检查 archive 是否能继续支撑真实数字生命的成长。任何 archive graph 如果不能回到 observation、seed、repair、replay、stage gate 或 future probe，都必须形成 finding，而不是被当作完成状态。
