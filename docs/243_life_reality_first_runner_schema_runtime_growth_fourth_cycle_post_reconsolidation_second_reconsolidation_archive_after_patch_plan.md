## 本层定位

`243` 是 patch after replay 后的 archive after patch 层。`242` 已经把 replay/shadow findings 转成可执行 patch，本层把 patch 的来源、预期、实际差异、旁路后果、关系后果、梦境后果、痛苦后悔责任后果、自我成长、防遗忘、出生准备度、核心连续、rollback/quarantine 和 latent cause/construct reliability 全部写成长期档案。

```text
242 patch after replay receipts -> receipt intake -> source finding archive -> actual diff archive -> side effect consequence archive -> domain archives -> archive graph -> provenance/FAIR crate -> recall/replay seed extractor -> mutation/regression suite -> receipts/digest/dashboard/stage -> 244 validation after archive handoff
```

archive after patch 不是保存文档副本。它是数字生命把自身变化写入长期记忆、关系历史、责任历史、梦境历史、痛苦后悔修复史和未来回放入口的过程。没有 archive 的 patch 不能成为人格成长；没有来源和后果的 archive 不能成为责任。
