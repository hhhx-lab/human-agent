# Bridge Tests

`tests/bridges/` 覆盖主体 slice 之后的桥接链：

- first activation preflight
- replay / shadow
- growth archive
- report bundle
- stage explanation
- digital life birth shell
- first terminal turn
- terminal life loop
- runtime growth

常用命令：

```bash
python3 -m unittest tests/bridges/test_runtime_growth.py -v
python3 -m unittest tests/bridges/test_first_terminal_turn.py -v
python3 -m unittest tests/bridges/test_terminal_life_loop.py -v
```
